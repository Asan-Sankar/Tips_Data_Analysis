# Restaurant Tips Data Analysis Using Python

This project performs exploratory data analysis on a restaurant tips dataset to understand tipping behavior. Statistical analysis and visualizations are used to study the impact of bill amount, party size, and customer demographics on tips. The insights help identify key patterns influencing tipping decisions in the restaurant industry.

## Installation by creating a python environment
To set up a Python environment for working with data analysis reports, follow these steps:

```bash
# Create a new virtual environment 
python -m venv .venv

#Activate the virtual environment
source .venv/Scripts/activate

# Create a requirements.txt file and add libraries inside

# Install required packages
pip install -r requirements.txt
```
